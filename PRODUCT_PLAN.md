# GEKO-AI Product Plan

## Mission
Unified AI platform with fully managed cloud/local models, no user API keys, freemium + premium SaaS.

## Revenue Model
- **Free:** Local models + free-tier cloud (no cost to us/user)
- **Pro ($29/mo):** All models, unlimited usage, auto-updates on new releases
- **PayGo:** Per-token billing (50% markup on provider costs)

## No API Key Management
Users NEVER provide or see OpenAI/Anthropic/Google keys. Platform manages 100%. 

## One-Click Local Models
Download Mistral → Click → App installs, runs, ready to chat (no CLI, no config)
